# Connecta

A social media platform built with vanilla HTML, CSS, and JavaScript that enables users to connect, share, and engage in real time.

## Overview

Connecta is a lightweight, browser-based social networking application designed for seamless user interaction. The platform provides a modern interface for creating posts, liking content, and commenting on shared updates. Built entirely with front-end technologies, Connecta operates entirely within the browser with local storage persistence.

## Features

- **Create Posts**: Share thoughts and ideas with an intuitive post creation interface
- **User Profiles**: Personalize your experience with custom usernames and avatars
- **Interactive Feed**: Browse, like, and comment on posts from other users
- **Real-Time Updates**: Posts update instantly across the application
- **Like System**: Express appreciation with a like feature
- **Comments**: Engage with content through threaded comments
- **Post Deletion**: Remove posts as needed with confirmation
- **Responsive Design**: Optimized experience across desktop and mobile devices
- **Local Storage**: Posts and user preferences persist across sessions
- **Animated Elements**: Smooth transitions and animations throughout the interface

## Technical Stack

- HTML5
- CSS3 with animations and gradients
- JavaScript (Vanilla - No frameworks)
- Browser Local Storage API

## Getting Started

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/prasannat05/Connecta.git
   ```

2. Navigate to the repository directory:
   ```
   cd Connecta
   ```

3. Open the application:
   - Open `index.html` directly in your web browser
   - Or use a local server (e.g., Python: `python -m http.server`)

## Usage

### Creating a Post

1. Enter your name in the username field
2. Type your message in the text area
3. Click "Post Now" to share

### Interacting with Posts

- **Like**: Click the heart icon to like a post
- **Comment**: Click the comment button to expand the comments section and add your reply
- **Delete**: Click the X button to remove your post

### Managing Your Profile

Update your username at any time - your avatar will update automatically based on the first letter of your name.

## Data Storage

All data is stored locally in your browser's local storage. This includes:
- User posts and content
- Like statuses and counts
- Comments
- Username preferences

Data persists across browser sessions and is not shared with external servers.

## Browser Compatibility

Connecta works best on modern browsers that support:
- ES6 JavaScript
- CSS Grid and Flexbox
- CSS Animations and Gradients
- Local Storage API

Compatible with Chrome, Firefox, Safari, and Edge (latest versions).
